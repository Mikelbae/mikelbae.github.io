# Michael de Greeve - Personal Portfolio Website

This repository contains the source code for my personal portfolio website, built from scratch and hosted live on GitHub Pages.

**View the Live Site:** [**mikelbae.github.io**](https://mikelbae.github.io)

---

## About This Project

This website serves as my professional portfolio, a personal sandbox for coding experiments, and a showcase of my skills in front-end development. It was built to be clean, interactive, and demonstrate a blend of professional presentation with personal creativity.

---

## Key Features

* **Cinematic Intro:** A "Click to Enter" screen that triggers a multi-second animated intro with sound effects, built with CSS animations and JavaScript.
* **"The Lab" Page:** An interactive playground for JavaScript-powered mini-tools, including:
    * An advanced D&D dice roller with support for multiple dice types, modifiers, and average-roll calculation.
    * A real-time text analysis tool (character, word, sentence counter).
    * A speed typing test.
    * A speech jammer (Delayed Auditory Feedback) experiment using the Web Audio API.
* **Easter Egg:** A hidden DVD-style screensaver easter egg (the Toto Wolff button) that triggers a "digital rain" effect and sound.
* **Responsive Design:** A clean, mobile-first layout that works across devices.
